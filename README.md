# Inventory Management System (Java)

A simple console-based Inventory Management System built using Java.

This project demonstrates Object-Oriented Programming principles, file handling, and CRUD operations using Java Collections.

---

## Features

- Add new products
- View all products
- Delete products by ID
- Save inventory data to file
- Load inventory data from file
- Input validation
- Basic error handling

---

## Technologies Used

- Java
- OOP (Classes, Encapsulation)
- ArrayList
- File I/O (BufferedReader, PrintWriter)
- Exception Handling

---

## Project Structure













- `Product.java` → Defines product object and file conversion logic.
- `Main.java` → Handles user interaction and business logic.

---

## How to Run

1. Clone the repository:









2. Open the project in IntelliJ IDEA.

3. Run `Main.java`.
